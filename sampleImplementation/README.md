# sampleImplementation
Sample app that exercises the framework end-to-end.

- `server/`: Sample controllers and server wiring.
- `scripts/`: Utility scripts (migrations).
- `data/`: Local runtime DB files.
- `client/`: Generated and custom wrapper client classes for sample ViewControllers.
- `models/`: Shared server/client model contracts used by generated clients.
- `db/`: Reserved for app-specific DB artifacts.
